# Player Metadata Source Consolidation Archive

This folder preserves historical migration evidence from the 2026-05 player
metadata consolidation.

Canonical player metadata now lives in:

- `data/metadata/projects/*/players.*.v1.json`

The archived `player_metadata.legacy_reference.v1.json` file is unverified
reference data only. The archived migration and review tools were used to move
approved SOOP IDs into project metadata and must not be reintroduced as active
pipeline commands.
