# Archive Docs

This folder is for historical notes that are useful for audit or context, but should not drive current implementation.

## Put Files Here When

- the file is a dated handoff memo
- the file explains a one-time incident or debug session
- the file contains planning notes that have already been executed or superseded
- the file is AI-specific operating guidance that no longer defines the live workflow

## Keep Out Of This Folder

- repo entry docs
- runbooks used in current operations
- active pipeline contracts
- reusable metadata contracts
