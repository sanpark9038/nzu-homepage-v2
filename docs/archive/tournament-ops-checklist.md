# HOSAGA Tournament Ops Checklist

## Before Event

- Verify roster metadata freshness
- Confirm schedule and ranking pages are rendering correctly
- Check latest pipeline status and alerts

## During Event

- Monitor admin tools for roster and match issues
- Confirm prediction and ranking flows behave as expected
- Track any source-site anomalies that require manual follow-up

## After Event

- Review pipeline outputs
- Sync approved data only after validation
- Record any operational issues for follow-up
