# NZU 대회 어드민 매뉴얼

본 문서는 NZU 홈페이지 대회 운영을 위한 어드민(`admin`) 경로와 각 페이지의 역할을 한 줄로 요약한 문서입니다.

| 경로 | 메뉴명 | 용도 |
| --- | --- | --- |
| `/admin/ops` | 운영 및 파이프라인 | 데이터 수집 파이프라인 수동 실행, 캐시 상태 및 운영 현황 모니터링을 담당합니다. |
| `/admin/roster` | 로스터 및 팀 관리 | 대회별 팀명 관리, 선수의 소속 팀/티어 수정, 비활성 선수 수집 예외 처리 등을 설정합니다. |
| `/admin/prediction` | 승부예측 설정 | 공식 경기 일정에 맞춰 승부예측(Match Voting) 경기 대진과 경기 시각(시간표)을 등록합니다. |
| `/admin/rankings` | 승패 및 순위 관리 | 진행된 대회 결과를 바탕으로 각 팀과 개별 선수들의 승/패 기록을 기입하여 순위표를 갱신합니다. |
| `/admin/data-lab` | 데이터 랩 | 전체 누적 데이터, 선수 및 종족별 통계 추세 등 확장 데이터를 조회 및 분석합니다. |
| `/admin/match` | 경기 기록 | 구체적인 단일 매치 및 전적 기록 에셋 관리 페이지입니다. |
| `/admin/login` | 관리자 로그인 | 어드민 페이지 접근을 보호하는 보안 인증 경로입니다. |
